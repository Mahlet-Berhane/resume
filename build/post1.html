My Cloud Computing School Project

Project Overview

In this project, I embarked on the exciting journey of building a personal website and a back-end API using modern cloud technologies. This experience was not just a learning curve but also a significant step towards standing out in the job market.

Benefits of the Project
  • Developed a personal website that impresses recruiters and distinguishes me from other candidates.
  • Gained hands-on experience with cloud technologies, enhancing my job interview and workplace performance.
Project Goals
  1. Build a fault-tolerant, highly scalable, production-ready API on the cloud.
  2. Create a front-end app that leverages the API, providing insights into full-stack development.
Tasks and Implementation

I tackled various tasks in this project, each contributing to the overall success and functionality of the application:

  • Developed the front end using HTML5, CSS3, js and deployed it on S3 with CloudFront.
  • Built a serverless back-end API utilizing AWS services such as Lambda, DynamoDB, and API Gateway.
  • Handled domain name acquisition and configuration, along with SSL certificate integration for security.
  • Implemented a resume website.
